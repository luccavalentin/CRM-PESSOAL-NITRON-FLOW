# Variáveis de Ambiente para Deploy

## Variáveis Necessárias

Adicione as variáveis de ambiente necessárias na plataforma de deploy (Vercel, Netlify, etc.).

## Como Adicionar

1. Clique em **"+ Add More"** para adicionar uma nova variável
2. Preencha o **Key** com o nome da variável
3. Preencha o **Value** com o valor correspondente
4. Repita para cada variável necessária
5. Clique em **"Deploy"** para aplicar as mudanças

## Importante

⚠️ **NUNCA** commite credenciais no código. Elas devem estar apenas nas variáveis de ambiente da plataforma de deploy.
